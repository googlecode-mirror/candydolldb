WELCOME:
--------
Welcome to CandyDollDB, a web based way of browsing, updating and
enjoying your CandyDollTV-collection.



DATE & VERSION:
---------------
2011-09-25, v1.3



NEWS:
-----
CandyDollDB has been downloaded almost 100 times! 



SYSTEM REQUIREMENTS:
--------------------
Computer running
 * PHP 5.3
 * MySQL 5.1
 * SMTP-mail server (optional)



INSTALLATION:
-------------
 * Extract the downloaded archive into a directory which is accessible 
through your web server.
 * Make sure the user running the web server has write-access to the 
CandyDollDB root directory.
 * Using your browser, navigate to setup.php and follow the on screen 
instructions. NOTE: Any previous CandyDollDB-database will be dropped.
